**POST /{timeRemaining}/tick** should replace the **#timer** element (i.e., itself) with a version that displays `Remaining time: {timeRemaining} seconds`, and after 1 second, triggers one of `POST /{timeRemaining-1}/tick`, `POST /close-block` or `POST /finish` depending on how much time is remaining and if there's text in the active textarea.

**POST /reset-timer** should simply return a newly initialised **#timer** element.

**POST /close-block** should replace the **#active-block** element with a new version, containing:

1. A read-only version of the previous active textarea.
2. A new active textarea.
3. A newly initialised **#timer** element.

**POST /finish** should take all the closed textareas and the active textarea, and join their contents together into a single read only textarea; the timer and finalise button should both be cleared too.

**Note:** because all the `<textarea>` elements share the name **"box"**, and are part of the same form, all POST requests in the parent `<form>` will have request body: `box=["{textarea1Content}", "{textarea2Content}", ... "{activeTextareaContent}"]`.
